# How do I work the funky gibbon?

Install go, clone the repo or whatever & run the following.

**Build the thing**
```
go build
```

**Use arguments..** 

```
./rollsay Hello, World
```

**..or use pipes**

```
echo "Hello, World" | ./rollsay
```

The bog roll will prefer arguments (first example) over piped data if arguments are provided.
